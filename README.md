## Getting Started
1. Clone the repository.
2. Install dependencies:
    - Frontend: cd frontend && npm install
    - Backend: cd backend && pip install -r requirements.txt
3. Start services:
    - Frontend: npm start
    - Backend: uvicorn app.main:app --reload
4. Run tests:
    - npm test
    - pytest

## Project Structure
- /frontend — UI code
- /backend — API and business logic
- /models — trained models and data artifacts
- /docs — design notes and meeting minutes

## Contribution Guidelines
- Use feature branches named feat/<feature> or fix/<issue>.
- Open PRs against main with descriptive titles and linked issues.
- Include unit tests and update docs when adding features.
